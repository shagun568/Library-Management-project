# Library Management System

This is a **Library Management System (LMS)** designed to help manage library operations efficiently. It allows admins and users to perform various tasks such as adding, updating, and deleting books, issuing and returning books, and generating reports.

## Features
- **User Roles:** Admin and User
- **Book Management:** Add, edit, delete, and search books
- **Issue/Return Books:** Track borrowed books and returns
- **Form Validations:** Ensures all required fields are filled
- **Access Control:** Admin can access all modules; users have limited access
- **Simple & Interactive UI:** Built with HTML, CSS, and JavaScript
- **Notifications:** Displays messages after successful operations

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Data Storage:** JavaScript objects/arrays (or localStorage for persistence)
- **Validation & Interaction:** JavaScript
